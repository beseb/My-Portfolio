## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/beseb/my-portfolio

# Access
$ cd my-portfolio

# Install dependencies with npm or yarn
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
